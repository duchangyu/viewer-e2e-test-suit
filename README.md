# viewer-e2e-test-suit

This is a suit of end-to-end test with Protractor. 

## Install and setup

* protractor needs node.js with higher version, greater than 4.2, I am using 5.1

* Install dependencies, java is needed, greater than 1.7
```    
npm install -g protractor
webdriver-manager update 
```


## Test gallery.autodesk.io sample

* Go to terminal window, go to the `ng-gallery` folder run following command 
```
webdriver-manager start (in another tab, Optional?)
protractor conf.js
```
